# projektpp

A simple C++ project.

## Requirements

- g++ (GCC C++ compiler with C++17 support)
- make

## Building

```bash
make
```

For debug builds with symbols:

```bash
make DEBUG=1
```

## Running

```bash
make run
```

Or run the binary directly:

```bash
./build/projektpp
```

## Cleaning

```bash
make clean
```
